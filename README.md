# MERN - CRUD

[LIVE DEMO](https://mern-crud-bg.netlify.app)

##FRONT - END
=============
CREATE `.env` File

```
VITE_APP_URL = YOUR_LOCAL_HOST_SERVER
```
(EXAMPLE : http://localhost:2000)

< -------------------------------------------------------------------------------------- >

##BACK - END
============
CREATE `.env` File

```
MONGODB_URL=mongodb+srv://(YOUR_USER_NAME):(YOUR_PASSWORD)@cluster0.noqkwj6.mongodb.net/crud
```
